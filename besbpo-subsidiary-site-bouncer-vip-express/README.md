# besbpo-subsidiary-site-bouncer-vip-express

Phase 4 pilot subsidiary site for Bouncer VIP Express (Pty) Ltd, forked
from `besbpo-subsidiary-site-template`. Implements `BESBPO-BLOG-ARCH-05`
Phase 4 (Subsidiary Pilot Wave).

- **Division:** `security-services` — new in this phase; see
  `besbpo-blog-architecture/adr/0003-add-security-services-division.md`
  for why the original 10-category taxonomy didn't have anywhere sensible
  to put this venture, and what was added.
- **Delivery mode:** `client_side`, `timeline` placement
- **Brand:** black + gold, matching the firm's existing
  Bronze/Silver/Gold/Platinum coverage-tier naming
