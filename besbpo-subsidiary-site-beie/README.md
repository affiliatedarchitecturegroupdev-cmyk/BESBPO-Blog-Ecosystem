# besbpo-subsidiary-site-beie

Phase 4 pilot subsidiary site for Bellwether Electrical & Instrumentation
Engineering (BEIE), forked from `besbpo-subsidiary-site-template`.
Implements `BESBPO-BLOG-ARCH-05` Phase 4 (Subsidiary Pilot Wave).

- **Division:** `construction` (secondary: `built-environment`)
- **Delivery mode:** `client_side`, `sidebar_widget` placement — this
  pilot deliberately tests a different placement than the Bellwether
  Architecture & Engineering pilot (`timeline`), per Doc-05 Phase 4's
  intent to validate the contract across placement variants, not just
  across divisions.
- **Brand:** green, black, white, navy (BEIE Nexus branding)
