# BESBPO Blog Ecosystem

A comprehensive multi-service blog platform built with modern technologies.

## Services

This monorepo contains the following microservices:

### Core Services
- **besbpo-blog-cms-api** - NestJS-based Content Management API
- **besbpo-blog-web** - Public-facing Next.js blog
- **besbpo-editorial-dashboard** - Next.js editorial dashboard

### Backend Services
- **besbpo-blog-syndication-svc** - Go-based RSS/Atom feed syndication
- **besbpo-blog-intelligence-svc** - Python-based AI content analysis
- **besbpo-blog-search-media-svc** - Rust-based full-text search
- **besbpo-blog-enterprise-svc** - Java enterprise integration

### Widgets
- **besbpo-embed-widget** - TypeScript embeddable widget

### Subsidiary Sites
- **besbpo-subsidiary-site-beie**
- **besbpo-subsidiary-site-bellwether-architecture**
- **besbpo-subsidiary-site-bouncer-vip-express**
- **besbpo-subsidiary-site-garlaws**
- **besbpo-subsidiary-site-lastmile-gig**
- **besbpo-subsidiary-site-template**
