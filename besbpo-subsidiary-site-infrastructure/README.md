# Infrastructure - BESBPO Subsidiary Site

## Overview

This is a subsidiary site for **Infrastructure**, part of the BESBPO Group blog ecosystem.

## Details

| Field | Value |
|-------|-------|
| **Site Name** | Infrastructure |
| **Subdomain** | infrastructure.besbpo.co.za |
| **Division** | infrastructure |
| **Locale** | en-ZA |
| **Description** | Strategic infrastructure development and project management |

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Run tests
npm test
```

## Deployment

This site is deployed via the main BESBPO CI/CD pipeline. See the infrastructure documentation for details.

## Content Structure

- Articles are managed through the main CMS API
- Content is syndicated from the primary blog
- Custom styling can be applied in `styles/globals.css`
